# My Next.js TypeScript Graphql

consume the data from [contentfulAPI](https://contentful.com/)

# Usage

```bash
yarn install
yarn dev
open http://localhost:3030
```
# Run test

```bash
yarn test
```

# check type

```bash
yarn check-types
```

# Technology

- Next.js 
- TypeScript
- jest
- GraphQL and Apollo Client
- Styled Components 
- ESLint
